# TikTok 데이터셋 샘플

<h2>1001개의 TikTok 프로필로 구성된 샘플 データセット</h2>

![TikTok dataset header](https://github.com/bright-kr/TikTok-dataset-samples/blob/main/tiktok-datasets.PNG)

1000개 이상의 프로필로 구성된 TikTok データセット 샘플입니다. データセット는 <b>Bright Data API</b>를 사용하여 추출되었습니다.

<h2>이 무료 データセット에 포함된 데이터 포인트:</h2>

* ```account_id```: TikTok 계정의 고유 식별자입니다
* ```nickname```: TikTok 사용자의 사용자 이름 또는 표시 이름입니다
* ```awg_engagement_rate```: 사용자의 콘텐츠에 대한 평균 참여율입니다
* ```comment_engagement_rate```: 사용자의 콘텐츠에서 댓글과 관련된 참여율입니다
* ```like_engagement_rate```: 사용자의 콘텐츠에서 좋아요와 관련된 참여율입니다
* ```predicted_lang```: 사용자의 콘텐츠에 대해 예측된 언어입니다
* ```is_verified```: TikTok 계정이 인증되었는지 여부를 나타냅니다(True/False)
* ```followers```: TikTok 사용자의 팔로워 수입니다
* ```following```: TikTok 사용자가 팔로우하는 계정 수입니다
* ```likes```: TikTok 사용자가 받은 총 좋아요 수입니다
* ```videos_count```: TikTok 사용자가 업로드한 총 동영상 수입니다
* ```id```: TikTok 사용자와 연관된 고유 식별자입니다
* ```url```: TikTok 사용자 프로필의 URL 또는 링크입니다
* ```profile_pic_url```: TikTok 사용자의 프로필 사진 URL입니다
* ```digg_count```: 사용자의 콘텐츠가 받은 총 "Digg" 상호작용 수입니다
* ```profile_pic_url_hd```: TikTok 사용자의 고화질 프로필 사진 URL입니다
* ```is_ad_virtual```: 사용자가 가상 광고와 연관되어 있는지 여부를 나타냅니다
* ```region```: TikTok 사용자와 연관된 지역입니다

그리고 더 많은 항목이 있습니다.

이는 "TikTok Profiles (public data)"
データセット에서 파생된 샘플 서브셋이며, 해당 データセット에는 <b>116,760,000개 이상의 프로필</b>이 포함되어 있습니다.

사용 가능한 データセット 파일 형식: <b>JSON, NDJSON, JSON Lines, CSV, 또는 Parquet. 선택적으로 파일을 .gz로 압축할 수 있습니다</b>.

データセット 전달 유형 옵션: <b>Email, API download, Webhook, Amazon S3, Google Cloud storage, Google Cloud PubSub, Microsoft Azure, Snowflake, SFTP</b>.

업데이트 빈도: <b>Once, Daily, Weekly, Monthly, Quarterly, 또는 Custom basis</b>.

추출된 데이터 포인트에 추가로 제공 가능한 데이터 보강: <b>요청에 따라 제공됩니다.</b>

<h2>TikTok データセット의 활용 사례는 무엇입니까?</h2>

<h3>1. Social Media Influence</h3>
TikTok データセット을 사용하여 강력한 소셜 존재감을 가진 인플루언서를 식별합니다. 이들의 참여도, 팔로워 기반, 브랜드 파트너십을 분석하여 브랜드 가치를 높이는 데 가장 적합한 대상과 협업할 수 있습니다.

<h3>2. Brand Monitoring</h3>
TikTok 데이터를 분석하여 브랜드, 제품 또는 서비스에 대한 대중의 정서를 파악합니다. 좋아요, 댓글, 공유, 해시태그, 멘션을 통해 인기 변화와 추세를 추적하여 트렌드에 앞서갈 수 있습니다.

<h3>3. Competitive Market Insights</h3>
경쟁 브랜드의 소셜 미디어 활동을 평가하여 경쟁 우위를 확보합니다. TikTok에서 해시태그, 게시물, 사용자 참여를 검토하여 전략을 고도화하고 경쟁사를 능가할 수 있습니다.
